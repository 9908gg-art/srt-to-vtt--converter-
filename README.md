# CaptionSafe: 免費線上 SRT 轉 WebVTT 字幕轉換器

這是一個專為影片創作者、YouTuber 與網頁前端開發人員設計的**高安全性、極速且高顏值**的線上字幕格式轉換工具。

專案採用純前端技術（HTML/CSS/JS）開發，可 **100% 免費部署於 GitHub Pages**，無須租用任何伺服器與資料庫。

👉 **[點此查看線上展示網站](https://你的使用者名稱.github.io/你的專案名稱/)** *(發布 GitHub Pages 後請修改此連結)*

---

## 🌟 核心特色功能

1. **100% 本地安全隱私保護**：
   * 採用瀏覽器 File API，檔案讀取與格式轉換**完全在使用者本地端進行，絕不上傳伺服器**。
   * 對於未公開、機密的影片字幕檔案，提供絕對的安全保障，即使在完全「斷網」的環境下也能完美運行。

2. **支援批次處理與拖曳上傳**：
   * 支援將多個 `.srt` 檔案直接拖曳進網頁進行大量批次轉檔。
   * 自動排程下載機制，能避開瀏覽器的多重檔案下載封鎖，實現一鍵批次打包下載。

3. **即時雙向狀態與預覽面板**：
   * 點選列表中的檔案，即可在右側/下方的「終端機風格」面板中即時預覽 WebVTT 格式的字幕內容。
   * 支援單檔直接下載與整體批次下載。

4. **霓虹玻璃擬態設計 (Neon Glassmorphism)**：
   * 針對影片創作工作者喜愛的暗色調，採用了高品質的暗紫羅蘭霓虹燈效與現代玻璃磨砂排版。
   * 完美適配手機與平板，出門在外用手機也能輕鬆完成字幕轉檔。

## 🛠️ 字幕格式轉換規則 (SRT ➡️ WebVTT)

工具會自動執行以下格式轉換以符合 W3C HTML5 標準：
1. **加入檔頭**：在檔案最開頭插入 `WEBVTT` 字樣及空行。
2. **時間格式修正**：將時間軸中的毫秒分隔符 **逗號 `,`** 取代為符合網頁標準的 **點號 `.`**。
   * *例如：`00:01:23,450 --> 00:01:25,900` 轉換為 `00:01:23.450 --> 00:01:25.900`*
3. **換行符清洗**：統一處理來自 Mac/Windows 不同系統產生的 `\r\n` 換行符。

## 🚀 如何快速部署到 GitHub Pages？

1. 在 GitHub 上建立一個新的公開儲存庫（Public Repository），例如命名為：`srt-to-vtt-converter`。
2. 將本專案的四個檔案上傳至該儲存庫：
   * `index.html`
   * `style.css`
   * `app.js`
   * `README.md`
3. 進入儲存庫的 **Settings（設定）** ➡️ **Pages**：
   * 在 Branch 下方選擇 **`main`**，然後點選 **「Save」**。
4. 約 1 分鐘後，你的專屬字幕轉換網站即在網際網路上發布成功！
